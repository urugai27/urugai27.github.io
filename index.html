さて、昨日の続きである。オリジナルの動画はSolveig_Song_yutube.mp4

1.音量レベルを確認

ffmpeg -i Solveig_Song_yutube.mp4 -vn -af volumedetect -f null -
結果
[Parsed_volumedetect_0 @ 000001b72db32d40] mean_volume: -46.2 dB
[Parsed_volumedetect_0 @ 000001b72db32d40] max_volume: -31.2 dB

max_volume:最大値が、-31.2 dBなので、この値を0 dbにしたい
ので、音量を31.2倍にすればいい、ただし、実際に変換後を確認すると０より
小さいので０にするために4（適当に）をさらに加えた

2.ゲイン調整（音量を３５倍:31+4)
→ 変換後のファイルはoutput_gein35_M-TRACK.mp4

ffmpeg -i Solveig_Song_yutube.mp4 -af volume=35dB
output_gein35_M-TRACK.mp4

音量変換後のレベルを確認

ffmpeg -i output_gein35_M-TRACK.mp4 -vn -af volumedetect -f null -
結果
[Parsed_volumedetect_0 @ 000001a3e92c2800] mean_volume: -12.6 dB
[Parsed_volumedetect_0 @ 000001a3e92c2800] max_volume: 0.0 dB

これで、動画の音量レベルを調整できたのでこのデータをユーチューブにアッ
プする。ソルヴェイグの歌
